# LMS

# Library Management System (LMS)

A fully functional Library Management System developed as a 2nd Year Group Project.

This system was built using MVC architecture and implements Role-Based Access Control (RBAC) to manage different user roles such as Admin, Staff, and Members.

---

## Tech Stack

- PHP (MVC Architecture)
- MySQL
- Bootstrap
- AJAX
- Role-Based Access Control (RBAC)

### Staff Features

## Role-Based Access Control

The system restricts access based on user roles:
- Admin
- Staff
- Member

Each role has specific permissions and access levels.

---

## Architecture

This project follows the **MVC (Model-View-Controller)** architecture:

- **Models** → Handle database logic
- **Views** → User interface
- **Controllers** → Application logic & request handling

## How to Run Locally

1. Clone the repository
2. Import the SQL file into MySQL
3. Configure database credentials in `main.php`
4. Run using XAMPP / WAMP
5. Access via `localhost/LMS`
